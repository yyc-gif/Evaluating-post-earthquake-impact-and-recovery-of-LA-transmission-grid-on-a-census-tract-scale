# Sensitivity Output Index

Use only the files in this folder for the cleaned sensitivity-analysis deliverable. All sensitivity calculations are restricted to the 2%-in-50-year (`2pc50`) scenario.

Regenerate these outputs through `C257H_Project_Main.py`. The main pipeline runs the live `2pc50` sensitivity experiment grid, including the three Stage 5 GA policies, and passes the resulting DataFrames to `build_sensitivity_outputs.py` for rendering. Each GA best order is generated by Stage 5 and then held fixed across the sensitivity cases. The renderer no longer treats `Data/sensitivity_summary_2pc50.csv` as an experimental input. The clean PNGs are generated without figure-level caption-style titles; those titles belong in the manuscript caption or table caption.

Before the first live run, execute `Topology_and_Weight.py` once. It exports the unthresholded tract-to-substation dependency matrix required to reconstruct each tested IDW threshold. The pipeline verifies that the production baseline, `theta_W = 0.03`, reproduces the production mapping before running the sensitivity grid.

## Supplementary

1. `Supplementary/Fig_Supp_IDW_Mapping_Diagnostics.png`
2. `Supplementary/Fig_Supp_Source_Gate_Mechanism_Diagnostics.png`
3. `Supplementary/Fig_Supp_Rank_Stability_T80.png`

Suggested captions:

Figure S1. Effect of the IDW dependency threshold theta_W on tract-dependency concentration. Increasing theta_W reduces the mean number of supplying substations per tract and increases the mean Herfindahl-Hirschman Index (HHI), indicating a shift from multi-source to more concentrated tract-level dependency. The blue axis shows mean supplying substations, and the red axis shows mean HHI.

Figure S2. Source-gate mechanism under the `2pc50` scenario. Using the common unconstrained physical-recovery reference, the left axis reports the time-averaged expected number of functional substations that remain connected to an active source over the modeled 0-480 h recovery period under each source-gate threshold tau_g. The right axis expresses the same quantity as a share of all modeled substations. The diagnostic is intentionally strategy-independent so that it isolates the source-gate mechanism from restoration-order effects.

Figure S3. Strategy-rank stability relative to the `2pc50` baseline configuration, measured using Spearman's rank correlation for population-weighted T80. Blank cells indicate parameter values not applicable to that sensitivity group. Average ranks are used for tied T80 values.

The full source-gate trajectories and their mechanism summary are exported to `Data/sensitivity_source_gate_diagnostics_2pc50.csv` and `Data/sensitivity_source_gate_mechanism_summary_2pc50.csv`.
